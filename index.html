hexo 部署

https://www.bilibili.com/video/BV1Yb411a7ty?p=1&t=235，预计一个小时完全搭建

博客日常使用步骤

  1. hexo n “name” 生成文章
  2. 在 source\_posts 中修改文章内容并保存
  3. clean、g、d 即可

注意

  1. 安装node.js、npm、cnpm、换源
  2. 修改 _config.yml 文件
  3. 设置 git 邮件用户名、ssh等
